## Problem
Beginners and college freshers often struggle to identify in-demand skills, reliable technologies, and industry expectations, causing confusion about career paths.

## Solution
SkillTrack is a web platform that guides users on skill development, trending technologies, and career opportunities. It suggests learning paths, skill stacks, roadmaps, and company opportunities based on skill sets.

## Features
- Input skills/courses with experience level  
- Compare against job requirements  
- Highlight skill gaps & match %  
- Suggest learning paths, mini-projects, and roadmaps  
- Recommend companies based on skills  
- Store profiles in LocalStorage  
- Explore trending skills

## Tech Stack
- HTML, CSS, JavaScript  
- LocalStorage (for profiles)  
- Future: Python backend + Database + AI-assisted guidance

## Future Aim
- Add Python backend for dynamic data handling  
- Integrate a database to store profiles and skill data  
- AI-assisted guidance for personalized recommendations, queries, and roadmap suggestions  

## Impact
Helps students make data-driven career decisions, reduces confusion, and improves employability.
